Item Dismantlement And Destruction

Made items in game can be dismantled or destroyed. Dismantling returns the components whereas destroying does not. The project effort is based upon the initial effort to craft in most cases with the exception of items containing metal which can take twice as long. They use the manufacturing skill, only allow the initiator to work on them, and provide EXP upon completion. Dismantling or destroying items in a place where vandalism is illegal will result in guard attacks.

Items that Cannot be Destroyed or Dismantled:

  • Resources
  • Bodies
  • Notes
  • Containers that have items in them


    • Items that can be Destroyed but Give No EXP:
  • Inventions without Components
  • Original Keys
  • God-Made Items


    • Items that can be Destroyed and Give EXP:
  • Non-Metal Tools
  • Inventions with Components
  • Food
  • Alchemy
  • Wooden Masks
  • Pens
  • Empty Containers


    • Items that can be Dismantled with Normal Effort:
  • Unlocked Locks
  • Key Copies
  • Any Item not Listed and not containing Metal


    • Items that can be Dismantled with x2 Effort:
  • Any Item containing Metal
